## 🚀 Featured Projects

### 🤖 [BU Life AI](https://bulife-ai.netlify.app) — Multi-Agent Campus Assistant
> *Jan 2025 – Present &nbsp;·&nbsp; LangGraph · FastAPI · Next.js · PGVector · Groq · NVIDIA NIM · Neon · Render · Netlify*

Campus queries are wildly heterogeneous — places, events, resources — and a single LLM call can't handle all of them efficiently. Built a LangGraph **supervisor that routes to specialized ReAct agents**, each with its own tool set and memory scope.

The parts that were actually hard:
- **Thread isolation** for concurrent user sessions without state bleed
- **Singleton retriever init** at startup — no per-request cold-start re-ingestion overhead
- **Hybrid BM25 + NVIDIA NV-EmbedQA 1024-dim vector search** — beats pure semantic on sparse keyword queries
- **SSE token streaming** end-to-end with full **LangSmith** observability on every trace

**Result:** 70% reduction in per-query LLM API calls. Fully deployed. It runs.
